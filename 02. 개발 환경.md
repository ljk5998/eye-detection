# 02. 개발 환경

## 1. 하드웨어 및 OS
*   **Device**: MacBook Pro (Apple Silicon, M1/M2/M3 Chip)
*   **Memory**: 16GB RAM
*   **OS**: macOS (Darwin)
*   **Camera**: 내장 FaceTime HD 카메라 또는 외부 USB 웹캠 (안구 중심 추적용 고해상도 입력 필요)

## 2. 프로그래밍 언어 및 런타임
*   **Language**: Python 3.12+ 권장 (Python 3.13은 패키지 호환성 확인 후 사용)
*   **Environment**: 가상환경 (venv 또는 Conda) 권장

## 3. 핵심 라이브러리 (Core Libraries)
| 라이브러리 | 버전 (권장) | 용도 |
| :--- | :--- | :--- |
| **MediaPipe** | 0.10.x | Face Mesh 기반 Iris Landmark 추출 및 홍채 중심점 좌표 획득 |
| **OpenCV (opencv-python)** | 4.10.x+ | 프레임 처리, 드로잉, 실시간 비디오 스트리밍 |
| **NumPy** | 2.x | 좌표 벡터 계산 및 통계 처리 |
| **Pandas** | 2.x | 시계열 안구 운동 데이터 저장 및 분석 |
| **Matplotlib** | 3.x | 이상치 시각화 및 운동 궤적 그래프 출력 |
| **SciPy** | 1.x | 신호 처리 (노이즈 제거, 속도 미분 등) |

## 4. 데이터 저장 구조
*   **Raw Data**: `.mp4` 또는 `.avi` 영상 파일
*   **Processed Data**: 홍채 중심 좌표값과 속도 지표가 포함된 `.csv` 또는 `.json` 파일
*   **Visualization**: 분석 결과가 포함된 `.png` 및 시각화 영상

## 5. 설치 방법 (CLI 예시)
```bash
# 가상환경 생성 및 활성화
python -m venv .venv
source .venv/bin/activate

# 핵심 패키지 설치
pip install -r requirements.txt
```

## 6. 개발 도구 (IDE)
*   **Primary**: VS Code / Cursor
*   **Debugging**: Jupyter Notebook (실험적 좌표 데이터 분석용)
