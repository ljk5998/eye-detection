# [Mini Project] 홍채 중심 기반 안구 운동 스크리닝 도구

## 1. 프로젝트 목적
본 프로젝트는 기존의 딥러닝 기반 사시/사위 분류(Classification) 방식에서 벗어나, **홍채 중심(Iris Center) 추적**과 **운동 역학 분석(Kinematic Analysis)**을 결합한 안구 운동 스크리닝 시스템 구축을 목표로 합니다.

학습 데이터에 의존하는 블랙박스 모델 대신, 물리적인 홍채 좌표 이동과 속도 변화를 측정하여 보다 객관적이고 설명 가능한(Explainable) 관찰 지표를 제시합니다. 본 도구는 의료적 진단을 내리는 시스템이 아니라, 안구 운동 이상 가능성을 탐색하고 기록하기 위한 보조 스크리닝 도구입니다.

## 2. 기존 방식과의 차이점
| 구분 | 이전 프로젝트 (U-Net/Transformer) | 새로운 프로젝트 (Geometric/Screening) |
| :--- | :--- | :--- |
| **핵심 기술** | End-to-End 딥러닝 (이미지 통학습) | 홍채 랜드마크 추출 + 벡터 분석 |
| **판단 기준** | 모델이 학습한 시각적 특징 (사시/정상) | 홍채 중심점의 이탈 방향, 거리, 속도 |
| **데이터 활용** | 대량의 레이블링 데이터 필요 (GT Mask) | 정상 안구 운동 범위 정의 및 편차 측정 |
| **주요 목표** | 사시/사위 유무 분류 | 안구 운동의 비정상적 패턴 후보 기록 및 시각화 |

## 3. 주요 기능 및 단계
1.  **정밀 안구 영역 검출**: MediaPipe Face Mesh를 활용하여 안면 및 안구 주변 랜드마크 추출.
2.  **홍채 중심점(Iris Center) 정의**: 좌/우 홍채 랜드마크의 중심 좌표를 실시간으로 산출.
3.  **운동 벡터 분석**:
    *   **Direction (방향)**: 기준점(정면 응시) 대비 홍채 중심이 이동하는 방향 분석.
    *   **Velocity (속도)**: 홍채 중심 이동 시 발생하는 미세한 흔들림 및 급격한 이동 속도 측정.
4.  **스크리닝 지표 산출(Screening Metrics)**:
    *   정상적인 안구 운동 범위와 편차 기준은 추후 실험 데이터로 정의하고, 1차 프로토타입에서는 좌표/속도/궤적 데이터를 우선 수집.

## 4. 기대 효과
*   **설명 가능성**: 단순 확률값이 아닌, "홍채 중심이 X축으로 몇 픽셀, Y초 동안 몇 픽셀의 속도로 이동했습니다"와 같은 구체적인 관찰 근거 제시 가능.
*   **유연성**: 특정 질환 진단이 아니라 안구 운동 패턴을 기록하고 검토하는 범용 스크리닝 도구로 활용 가능.

## 5. 기술 스택 (예정)
*   **Language**: Python 3.12+ 권장
*   **Computer Vision**: OpenCV, MediaPipe (Iris Landmark)
*   **Data Analysis**: NumPy, SciPy, Pandas
*   **Visualization**: Matplotlib, Streamlit (실시간 모니터링용)
